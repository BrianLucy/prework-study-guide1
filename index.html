Git work flow

  1. Working directory
  2. Staging area
  3. Local repository
  4. Remote repository
  • git status: checks what branch we are currently on
  • git clone creates a repo clone on local machine
  • git init creates or initializes a new git project
  • git pull origin master pulls the repo from github
  • git log shows all commit logs for the project
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add adds changes to staging area
  • git commit -m: creates a new commit with a log messagedescribing the changes to the repo
  • git push pushes changes to the repo on github
  • pull request asks to merge changes in to main branch
